# Profile Setup

This folder is a ready-to-publish GitHub profile README scaffold for the `svenbuild` account.

To make it appear on the GitHub profile page:

1. Create a public repository named `svenbuild`.
2. Copy `profile/README.md` to the repository root as `README.md`.
3. Copy `profile/pages/skills.md` and `profile/pages/stats.md` into a `pages/` folder in that repository.
4. Commit and push.

GitHub only shows the profile README when the repository name exactly matches the username.
